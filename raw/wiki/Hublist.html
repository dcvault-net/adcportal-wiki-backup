Description

A Hublist is used for listing online public hubs to the user upon request from the client, older versions of hublists (NMDC) had uncompressed hublists in the beginning this was replaced by BZIP compressed lists.

Another part is the pinger in older hublists registration servers where used what needed pinger information to be entered these days pinger gets information from SQL databases in order to connect to hub.
