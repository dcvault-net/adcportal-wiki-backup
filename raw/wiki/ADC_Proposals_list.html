ADC Proposals list

Information Related Information
These extensions are proposed to be included in future versions of ADC; their status is currently drafting. Please update this list if any of the specifications in draft are added or denied entry to the protocol, start with newest first and oldest at the bottom.

Too get your idea listed here post your idea at ADCPortal in the Protocol Ideas section.

AUT0 - Network connection auto detection extension Dj Offset Detect if client is capable of initiating active connections. Extensions Proposal 2011 Link to discussion
CIDM - Client implementation debug mode Pretorian Allow clients to debug hub sessions without being kicked for invalid messages etc. Extensions Proposal 2011 Link to discussion
CCPM - Client to Client Private Message Pretorian Allow C-type MSG. Extensions Proposal 2011 Link to discussion
IPv6 RFC Update darkKlor Updating the RFC number for IPv6 to the latest draft standard Base Proposal 2011 Link to discussion
CHCM Flipflop clickable hubcommands in a chatline. Extensions Proposal 2010 Link to discussion
TRNR CyB Download and upload rate calculation and notification. Extensions Proposal 2010 Link to discussion
SUGR Flipflop grouping of different supported features that are known to occur together frequently, can save bandwidth. Extensions Proposal 2010 Link to discussion
