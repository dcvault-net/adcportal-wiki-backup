ADC is a decentralized network , thus we have hubs acting as hyper network nodes. Hubs allow a certain number of clients to connect, but they don't intercommunicate. This allows creating private networks as well as public networks that gets listed at hublist sites at owners request. Some softwares are more evolved then others some have more feature control while some have a visual interface that allows editing or a plugin/script interface that allows the user to make custom plugins/scripts for their hub. [1][2][3] If a hubsoft doesnt have a plugin system there are also bots that could be added in the hub to fill up functions.
