Correctness of implementations

It may be good to remove all the implementations listed in the last part of the article. As there is a good chance that none of these implementations implements the here described Bloomfilter... Especially they might not even implement any to each other compatible Bloomfilter. I checked at least the Java implementation and thats not an implementation of the Bloomfilter described in the spec. --Quicksilver 15:32, 10 August 2009 (UTC)
