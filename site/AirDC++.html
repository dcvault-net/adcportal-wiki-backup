Introduction

AirDC++ is a Direct Connect Client mod based on StrongDC++ and DC++. AirDC++ tries to cope with the requirements that regular users ask into making a client for the taste of everyone.

Features

AirDC++ uses a nice tabbed interface where the tabs can change colour instead of the traditional bolding that makes tabs jump all over. It has several settings to customize the refresh interval, both on start up and for selected folders (a new feature that will be introduced in 2.08). Advanced ADL searches using reg exp and comments to make it easier to keep track of things. Keeps shared paths in settings even if they are not available on start up (enhancements for networked share and external drives). AirDC++ provides one skip list for share, and one for download, and introduced in 2.06 there is also the possibility to have predefined skip lists for searches. Highlight chat based on patterns or reg exp, make your own or use any of the built in definitions. Uses colours in file lists and searches to clearly mark files that are already in share to avoid unnecessary downloading. The GUI can be locked with a password when minimized, and there are many other AirDC++ specific add-ons that you will not find in other clients.

Updated language files are automatically detected and installed using the update dialogue (currently there are ten actively supported languages, more are added for each version). We are always looking for new people that is interested to help translating.

Starting with version 2.08 there will be a new installer process that we hope will work much better for all of you that uses Windows Vista or Windows 7 and finally solving the annoying issues with those new operating systems write protecting the program files directory.

History

AirDC++ 1 was first released early in 2007 as a continuation of the closed fulDC project and had a few releases during the year until it was finally decided that it would be better to use the new core to take advantage of the work that had been done to it instead of trying to fix all the problems in the old fulDC core.

The first version of AirDC++ 2 was released in December 2007, this was the first AirDC++ with the new core from StrongDC++ supporting segmented downloading and many security enhancements, since then there has been a new release each 3 - 4 month with many improvements and new functions.
